API documentation for gxp, to be viewed at http://opengeo.github.com/gxp.

To create or update this documentation, run ant in the build directory of the
master branch of this repository, and copy the content from build/gxp/doc/ to
this directory. Don't forget the empty .nojekyll file in the root of this
branch that tells github to not use Jekyll. If this file is missing, the
_static directory will not be published.
