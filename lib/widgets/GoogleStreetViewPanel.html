Config Options¶

Configuration properties in addition to those listed for Ext.Panel.

heading
Number The camera heading in degrees relative to true north. True north is 0 degrees, east is 90 degrees, south is 180 degrees, west is 270 degrees.
location
OpenLayers.LonLat The panorama location
pitch
Number The camera pitch in degrees, relative to the street view vehicle. Ranges from 90 degrees (directly upwards) to -90 degrees (directly downwards).
zoom
Number The zoom level. Fully zoomed-out is level 0, zooming in increases the zoom level.
