GXP Basics
==========

`GXP <http://github.com/opengeo/gxp>`_ is a young, rapidly-developing library for
building rich, web-based GIS applications. The library is built upon `GeoExt <http://www.geoext.org>`_, 
`Ext JS <http://extjs.com/>`_ and `OpenLayers <http://openlayers.org/>`_. 

.. rubric:: What this module covers

In this module you will create a viewer application using readyGXP, dissect your viewer to understand the parts, and get links to additional learning resources.
