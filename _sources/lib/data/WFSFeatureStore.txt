Config Options
--------------

Configuration properties in addition to
those listed for `Ext.data.Store <http://extjs.com/deploy/dev/docs/?class=Ext.data.Store>`_.


.. describe:: maxFeatures

    ``Number``
    Optional limit for number of features requested in a read.  No limit
    set by default.

.. describe:: ogcFilter

    ``OpenLayers.Filter``
    Optional filter to set on the WFSProtocolProxy.






Public Methods
--------------

Public methods in addition to those
listed for `Ext.data.Store <http://extjs.com/deploy/dev/docs/?class=Ext.data.Store>`_.


.. method:: WFSFeatureStore.setOgcFilter

    :arg ogcFilter: ``OpenLayers.Filter`` Update the filter used by the
        protocol proxy.  You must manually call load or reload to trigger
        loading.
